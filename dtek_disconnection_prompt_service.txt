Мне нужен Python + Playwright скрипт, который вызывает https://www.dtek-dnem.com.ua/ua/shutdowns

Вначале делает проверку, если есть div с class="modal__container m-attention__container #modal-attention > div > div, то нажми на <button class="modal__close m-attention__close" aria-label="Close modal" data-micromodal-close=""></button>, а уже только потом приступает к основному сценарию.

Вносит в следующие поля следующие данные:
input #city=м. Дніпро
input #street=вул. Сонячна Набережна
input #house_num=6

Это значения по умолчанию.

Необходимо предусмотреть возможность, чтобы эти значения #city, #street и #house_num могли бы 
передаваться извне в качестве параметров при запуске этого скрипта в качестве сервиса.

Они могут передаваться в качестве HTTP-параметров или, например, телеграм-ботом.

Сделай это с помощью argparse, чтобы вызов функции выглядел бы, например, вот так:
python dtek_disconnection_gemini.py --city "м. Новомосковськ" --street "вул. Бойова" --house "11"

Сохраняет данные #discon-fact > div.discon-fact-tables в JSON.

Заполнение значений должно быть небыстрым, потому что скрипт может работать нестабильно.

Параллельно сохраняет screenshot div class="discon-fact active" в файл discon-fact.png

Сайт https://www.dtek-dnem.com.ua/ua/shutdowns — это динамический веб-сайт на JavaScript (скорее всего, на React или аналоге).

JSON файл имеет следующую структуру:

[
  {
    "city": "м. Дніпро",
    "street": "вул. Сонячна Набережна",
    "house_num": "6",
    "group": "3.2",
    "date": "02.11.2025",
    "slots": [
      { "time": "15 – 16", "disconection": full },
      { "time": "16 – 17", "disconection": full },
      { "time": "17 – 18", "disconection": half },
    ]
  }
]

Значение "city" берется из value of #discon_form #city
Значение "street" берется из value of #discon_form #street
Значение "house_num" берется из value of #discon_form #house_num

Значение "group" берется из "#discon_form #group-name > span"

Значение "date" берется из #discon-fact > div.dates > div.date.active > div:nth-child(2) > span

Значения "time" берутся из #discon-fact > div.discon-fact-tables > div.discon-fact-table.active > table > thead > tr > th:nth-child(2) > div по #discon-fact > div.discon-fact-tables > div.discon-fact-table.active > table > thead > tr > th:nth-child(25) > div, причем только из тех, где #discon-fact > div.discon-fact-tables > div.discon-fact-table.active > table > tbody > tr > td имеет class="cell-scheduled" или class="cell-first-half" или class="cell-second-half".

Значения "disconection" берутся из #discon-fact > div.discon-fact-tables > div.discon-fact-table.active > table > tbody > tr > td
причем только тех, где #discon-fact > div.discon-fact-tables > div.discon-fact-table.active > table > tbody > tr > td имеет class="cell-scheduled" или class="cell-first-half" или class="cell-second-half":

если td имеет class="cell-scheduled", тогда "disconection": full,
еесли td имеет class="cell-first-half" или class="cell-second-half", тогда "disconection": half.

После ввода значений в поля, внизу появляется список найденых значений. Нужен первый элемент. После ввода нужно выбрать первое значение из списка и нажать Enter.

Для названия города список имеет div #cityautocomplete-list,
для названия улицы #streetautocomplete-list,
для номера дома #house_numautocomplete-list.

автодополнением с точным совпадением
парсингом и JSON

<p><span lang="UK">Стабілізаційні відключення відбуваються згідно із визначеним графіком на добу. Максимальна тривалість одноразового відключення – 7 годин. Екстрені відключення відбуваються раптово, коли дуже швидко треба збалансувати енергосистему.</span></p>
<p><span lang="UK">Саме НЕК Укренерго визначає, який тип відключень і коли потрібно застосовувати та передає відповідну команду всім ОСР.</span></p>
<p><span lang="UK"><strong>Як зрозуміти, чи я маю аварійне відключення чи діють обмеження споживання?</strong></span></p>
<p><span lang="UK">Інформація про будь-які відключення публікується на сайті компанії. Введіть вашу адресу та дізнайтеся про поточні відключення. Також на головній сторінці нашого сайту ми оперативно розміщуємо інформацію про те, чи діють зараз графіки стабілізаційних відключень, або застосовуються екстрені відключення.</span></p>
<p><span lang="UK"><strong>Чому важко додзвонитися до КЦ? Куди я можу повідомити проблему чи поставити своє запитання?</strong></span></p>
<p><span lang="UK">Під час екстрених відключень ми справді фіксуємо величезну кількість дзвінків до контакт-центру, яка більше за звичайну у 6-10 разів. Дуже часто, під час екстрених відключень, відбуваються повітряні тривоги і наші співробітники йдуть у бомбосховище, щоб продовжити працювати вже звідти, але їм потрібен час на переміщення. Через це зростає тривалість очікування на лінії. Однак ви завжди можете повідомити про проблему чи дізнатися актуальну інформацію на сайті чи на сторінці&nbsp;<a href="https://www.facebook.com/dtekdnem">Facebook</a>.</span></p>
<p><span lang="UK"><strong>Чому за моєю адресою екстрені відключення, якщо вже давно не було обстрілів?</strong></span></p>
<p><span lang="UK">Ворог завдав величезної шкоди енергосистемі і зараз не полишає спроб лишити нас без світла. Енергетики працюють не покладаючи рук, щоб ремонтувати, посилювати та забезпечувати країну світлом. Однак коли споживання надто зростає, щоб вберегти енергосистему від масштабних аварій доводиться проводити відключення. Ми розуміємо, наскільки важко вам знаходитись без світла. Але просимо вас набратись розуміння та вірити в енергетиків, ми робимо все, аби відключення були якомога коротшими.</span></p>
<p><span lang="UK"><strong>Навіщо превентивні відключення світла під час обстрілів?</strong></span></p>
<p><span lang="UK">Це потрібно для того, щоб у випадку руйнувань енергооб'єктів через обстріли, запобігти більш масштабним аваріям. Адже якщо такий об'єкт на повній потужності в один момент буде виведено з ладу, це спричинить різке розбалансування енергосистеми. Відключення відбуваються за командою НЕК Укренерго, коли він отримує попередження від військових про потенційну атаку на енергоінфраструктуру. Якщо загроза зникає, після відбою тривоги світло повертається.</span></p>
<p><span lang="UK"><strong>Чому світло вимикають вночі?</strong></span></p>
<p><span lang="UK">Обмеження споживання діють як вдень, так і вночі. Тому що попри зменшення навантаження вночі - споживання електроенергії все рівно продовжується, особливо в холодний період. Нічні обмеження менші, ніж денні, однак вони також потрібні, щоб не сталося аварій на електромережах та підстанціях.</span></p>
<p><span lang="UK"><strong>Чому пошкоджені мережі так довго ремонтують?</strong></span></p>
<p><span lang="UK">За рік енергетики відремонтували всі ключові енергооб'єкти генерації та електромереж. Зробили все можливе для того, аби пройти цю зиму без відключень. Однак ворог не полишає своїх спроб лишити українців без світла і продовжує атакувати станції генерації та магістральні мережі, і це може призвести до необхідності відключень. Проте вірте в енергетиків і воїнів ППО, ми робимо все, аби уникнути відключень!</span></p>
<p><span lang="UK"><strong>Чому електроенергію продають за кордон, коли ми сидимо без світла</strong>?</span></p>
<p><span lang="UK">Відключення відбуваються для запобігання аварійних ситуацій в енергосистемі. Україна не продає електроенергію, яка потрібна українським споживачам.</span></p>
<p><span lang="UK"><strong>Чому світло вимикають і коли холодно, і коли жарко?</strong></span></p>
<p><span lang="UK">Бо саме в ці періоди ми, споживачі, починаємо користуватись більшою кількістю електроприладів, щоб почуватися комфортніше. Взимку це електроопалення, влітку – кондиціонери. Ця техніка є енергозатратною – 1 кВт і вище. І коли ми вмикаємо її одночасно з іншими приладами, то перевантажуємо мережу. Якщо в такому випадку не обмежити споживання електрики, ослаблена енергосистема не витримає – і виникнуть масштабні аварії. Тоді нам доведеться сидіти без світла значно довше.</span></p>
